# AI Industry Impact Analysis (NLP Project)

## 1. Background

This project analyzes how artificial intelligence is impacting different industries and companies based on a large-scale news dataset.

## 2. Objective

* Identify which industries are most impacted by AI
* Understand how AI affects companies (positive / negative / mixed)
* Explore key technologies driving this impact

## 3. Data

* ~200,000 news articles
* Fields include title, text, companies, industry, and publication date

## 4. Methodology

* Topic Modeling (LDA)
* Named Entity Recognition (spaCy)
* Sentiment Analysis (fine-tuned model)
* Aggregation at industry and company level

## 5. Key Results

* AI impact is concentrated in media, finance, education, and tech
* Major companies: Microsoft, Google, Amazon, Nvidia
* Technologies: Generative AI, cloud computing, enterprise software

## 6. Insights

* AI adoption is strongest in software-driven industries
* Impact is more uncertain in regulated sectors like healthcare
* Companies with strong infrastructure benefit the most

## 7. Tech Stack

Python, pandas, scikit-learn, spaCy, matplotlib
